# Conecta

Sistema desenvolvido para unir comunidades em eventos públicos nas praças de Florianópolis, promovendo atividades físicas e culturais para crianças, adolescentes, adultos e idosos.

---

# Problema/Desafio

O Conecta é um aplicativo que une comunidades em eventos públicos nas praças de Florianópolis. O objetivo é incentivar crianças, adolescentes, adultos e idosos a praticarem atividades físicas e culturais, criarem laços sociais e se desconectarem das redes sociais.

O projeto busca combater problemas como:
- Sedentarismo
- Vícios digitais
- Fobias sociais
- Falta de interação comunitária

Além disso, promove:
- Liberdade de expressão
- Diversão
- Bem-estar físico e mental
- Apoio a profissionais autônomos

# Instruções de Execução

## 1. Clonar o repositório

bash
git clone LINK_DO_REPOSITORIO


## 2. Entrar na pasta do projeto

bash
cd nome-do-projeto


## 3. Instalar as dependências do back-end

bash
npm install


## 4. Configurar o banco de dados PostgreSQL

- Criar o banco no pgAdmin
- Executar o arquivo SQL do projeto

## 5. Iniciar o servidor

bash
npm start


ou

bash
npm run dev


## 6. Abrir o sistema

Abrir o front-end no navegador ou acessar:

bash
http://localhost:3000
